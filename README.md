# chartjs-ror

Simplifies using [Chart.js][] in Rails.

## Installation

Add this line to your application's Gemfile:

    gem 'chartjs-ror'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chartjs-ror
